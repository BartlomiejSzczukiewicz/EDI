Brief Introduction

Fictional shop - Electronic data interchange project, is a fresh website of a pretend e-shop. The site uses special font for comfortable reading, used within the sidebar. Sidebar contains of easy-to-navigate links leading to the Home, Data, and Authors pages. Here are the main things about the website: 1. Sections: There are detailed sections that tell you everything about the clients - what each person bought and how much quantity 2. Easy to Use: The website is designed to be easy to use. You can move around and find things without any trouble. In simple terms, this project is an example of a website that is easy to use, and gives you all the info you need about the store and its team.
